## ACG-face-master

### 1 所需依赖库
* python 3.6
* pytorch 1.3.0
* torchvision 0.4.1
* opencv-python 3.4.1



### 2.开始使用
1. 下载 [模型文件](https://pan.baidu.com/s/1Mw9aulKGuFJesIsbcFfxhQ) 并放入 ``` checkpoints/weights ``` 。 
2. 运行 ``` python v.py 视频文件路径 图片文件路径 ```
3. 示例:
https://www.bilibili.com/video/BV1AL411A7W1/
